# Complete Genogram Symbols Reference

## Individual Symbols

### Gender
| Symbol | Unicode | Meaning |
|--------|---------|---------|
| □ | U+25A1 | Male |
| ○ | U+25CB | Female |
| ◇ | U+25C7 | Non-binary/Unknown |

### Status Modifiers
| Modifier | Meaning |
|----------|---------|
| X through symbol | Deceased |
| Slash through symbol | Stillbirth |
| Dotted outline | Pregnancy |
| Triangle | Miscarriage |
| Small filled triangle | Abortion |

### Special Cases
| Symbol | Meaning |
|--------|---------|
| Concentric squares/circles | Index person (IP) |
| Double border | Identified patient |
| Dashed outline | Adopted |

### Child Types
- Biological (solid vertical line)
- Adopted (dashed vertical line with A)
- Foster (dotted vertical line with F)
- Stillbirth (slash through child symbol)
- Twins (diagonal lines from same point)
- Identical twins (diagonal lines with connecting bar)
